# 简历下载按钮设计

## 目标

为个人网站增加两个“下载简历”入口，让访客可以直接下载黄佳沐的 AI 产品经理简历 PDF，同时保持网站现有黑白、半透明、克制的视觉风格。

## 方案选择

采用站内静态文件直链方案：将 PDF 文件放入 `public/resume/`，两个按钮均使用带 `download` 属性的链接直接下载文件。

未采用下载 API，因为当前网站没有统计系统，引入 API 会增加不必要的复杂度。未采用外部云盘链接，因为跳出网站会削弱下载体验。

## 界面设计

### 首屏入口

在首页首屏姓名与个人介绍下方增加“下载简历”按钮。按钮沿用网站已有的黑白玻璃质感：半透明背景、细边框、轻微阴影与悬停反馈。

首屏介绍容器目前禁止指针事件。新增按钮时，仅为按钮恢复指针事件，避免影响其他首屏元素。

### 联系区入口

在联系区已有“联系我”和“查看 GitHub”按钮旁增加“下载简历”按钮，复用现有 `.btn` 与 `.btn-ghost` 样式。

移动端继续沿用联系区按钮纵向铺满的布局规则。

## 文件与链接

- 静态文件：`public/resume/黄佳沐-AI产品经理.pdf`
- 下载链接：`/resume/黄佳沐-AI产品经理.pdf`
- 下载文件名：`黄佳沐-AI产品经理.pdf`

## 验证

- 运行 `npm run lint`。
- 运行 `npm run build`。
- 在浏览器中检查桌面端和移动端布局。
- 点击两个入口，确认链接指向站内 PDF 且带有下载语义。
