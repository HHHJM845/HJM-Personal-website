# 个人网站信息更新设计

## 目标

根据新版《黄佳沐_AI产品经理》简历，同步个人网站首页信息，并更新站内可下载简历文件。

## 方案选择

采用「完整内容同步，保留现有视觉框架」方案。首页继续沿用现有单页结构、黑色动态背景、项目卡片、能力标签和时间线，只替换与新增简历中的关键信息。

未采用重新设计页面结构的方案，因为这次目标是让公开信息及时准确；现有站点已经具备较完整的叙事节奏和动效资产，重做版式会扩大风险。

## 信息架构

页面更新为新版定位：AI 产品经理，聚焦 AI 工作流、RAG、机器视觉质检。核心内容包括：

- 顶部数据：新增机器视觉质检指标，保留内容创作与 RAG 业务成果。
- About：强调 3 年 AI 产品与数字化产品实践经验。
- Projects：加入 AI 视觉质检系统，并同步 AI 内容协作平台、RAG 智能客服与销售培训系统、网易有道教育 RAG 经历。
- Capabilities：覆盖 AI 产品设计、用户场景与商业化、数据评测、工具原型。
- Experience：更新深圳跃然像素科技结束时间为 2026.05，并保留深圳华禧文化、网易有道、广州美术学院经历。

## 验证

- 用内容校验脚本确认关键文案、指标和新版简历链接存在。
- 运行 `npm run lint`。
- 运行 `npm run build`。
