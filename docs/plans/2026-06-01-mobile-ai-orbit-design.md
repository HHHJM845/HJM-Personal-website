# 手机端 AI 工具轨道卡片优化设计

## 目标

优化 About 区域 AI 工具轨道卡片在手机端的视觉表现，保留全部 17 个工具图标，同时解决中心拥挤、底部堆叠、外圈留白不均和卡片占位偏大的问题。

## 方案

桌面端保持现状。仅在小于 `768px` 的手机端使用专用坐标和尺寸：

- 为每个工具增加手机端坐标与尺寸。
- 图标重新分布到外环、内环和少量中心邻近位置。
- 整体图标尺寸缩小约 15%-25%。
- 卡片宽度跟随屏幕，保持接近正方形并居中。
- 适当缩小中心标识，降低视觉抢占。
- 保留现有轻微浮动动画与工具链接。

## 验证

- 运行 `npm run lint`。
- 运行 `npm run build`。
- 使用手机端视口检查图标分布、卡片尺寸与页面位置。
- 使用桌面端视口确认原有布局未受影响。
